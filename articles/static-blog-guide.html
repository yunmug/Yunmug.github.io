静态博客搭建指南

很多人觉得搭博客需要学 React、Vue 或者 WordPress。其实完全不必。这篇文章介绍如何用纯静态文件搭建一个轻量博客，并通过 GitHub Pages 免费部署。

目录结构

yunmug.github.io/
├── index.html          # 首页
├── articles/
│   ├── manifest.json    # 文章元数据
│   └── hello-world.html # 各篇文章

核心思路

首页通过 JavaScript 异步加载 articles/manifest.json，获取所有文章的标题、日期、摘要和路径，然后渲染成卡片列表。

发布新文章的流程非常简单：

1. 在 articles/ 下写好 HTML 文件
2. 在 manifest.json 里新增一条记录
3. Push 到 GitHub，等待 Pages 自动部署

没有数据库、没有构建步骤、没有任何依赖。纯 HTML/CSS/JS，所有浏览器都能直接打开。
